**FluentUI TagList PCF** Control that turns a dataset into a Tag list (FluentUI Badges).

## Parameters
| Parameter         | Description                                                                                  | Default     |
|-------------------|----------------------------------------------------------------------------------------------|----------   |
| Tags Dataset  | Dataset to Expose                                                             |             |
| -> Tag Label    | Text of the tag (Dataset Bound Column)                                                         | |
| Show Record Image   | Show the record image beside the text.                                                         | false   |
| Open Record OnClick    | Open Record OnClick                                                        | false     |
| Appearance   | Appearance of the Badge.                                                              | filled        |
| Size| Size of the Badge.    | Medium      |
| Shape| Shape of the Badge.   | Rounded      |
| ImageShape| Shape of the Image if showRecordImage is set to true.   | Rounded      |
| Color| Color of the Badge.   | Brand      |
| Theme| Theme   | Web Light      |
